HtmlGraph Product Specification

Draft Author: Shakes Created: 2025-12-21

Overview

Build a zero-dependency graph database using HTML files as nodes, hyperlinks as edges, and CSS selectors for queries

Context

AI agent developers need simple coordination infrastructure without Docker/databases. HtmlGraph provides human-readable, git-friendly agent state management

Acceptance Criteria

  1. ⏳ Parse 1000 HTML nodes in <1 second test_performance_large_graph
  2. ⏳ CSS selector queries return results in <100ms test_query_performance
  3. ⏳ Dashboard loads and renders in <500ms test_dashboard_load_time
  4. ⏳ Zero external dependencies except justhtml + Pydantic test_dependencies
  5. ⏳ Works offline without network access test_offline_mode
  6. ⏳ Git diff shows clear changes to graph state test_version_control_friendly
